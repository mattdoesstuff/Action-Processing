# Action-Processing

Built during the HackHSN Hackathon in New Jersey.

This app was built after the International Strike for Climate with an aim to raise awareness and action towards climate responsibility.

The app encourages users to commit simple daily tasks that help our enviroment.
A tree will be slowly grown based on the number of tasks completed.
Action! also shows a heatmap calendar that reflects past task completion history.

This app was built in Processing 3.0 in Java as a proof of concept.
To run open the action.pde file using the Processing app.
